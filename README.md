# 📖 Overview

**HireHub** is a full-stack Recruitment Management System developed using the **MERN stack**. It provides a complete recruitment workflow where recruiters can manage job postings and applications, while candidates can search, apply, and track their recruitment status through dedicated dashboards.

The platform focuses on providing a modern, scalable, and user-friendly hiring experience with role-based access control and premium dashboard interfaces.

# ⚙️ Installation

## Clone the repository

```bash
git clone https://github.com/shwetankraj04/hirehub.git

cd hirehub
```

---

## Backend Setup

```bash
cd server

npm install

npm run dev
```

---

## Frontend Setup

```bash
cd client

npm install

npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the `server` directory:

```env
PORT=

MONGO_URI=

JWT_SECRET=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```
